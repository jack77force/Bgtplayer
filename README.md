2) curl https://raw.githubusercontent.com/creationix/nvm/master/install.sh | bash

3) source ~/.bashrc

4) nvm install node

5. Clone the Repository :
git clone https://github.com/Monu0777/Maxmusic &&  cd DevilPlayer

pip3 install -r Installer

vi Internal
